## 本项目和线一的关系

| | 线一（RF_Image_GAN） | 线二（RF_Image_Bmode） |
|---|---|---|
| **核心思路** | 纯 RF → B-mode | RF + 低质量 B-mode 双输入 → 高质量 B-mode |
| **灵感来源** | 条件 GAN（pix2pix 范式） | Hy-PCF 论文（双输入跨域融合） |
| **训练机器** | Linux 服务器（RTX 5060） | Win11 WSL（RTX 4060） |
| **代码仓库** | Linux: `/media/liujia/8CC24D13C24D02C6/code/RF_Image_GAN/` | WSL/Windows: `G:\code\RF_Image_Bmode\` |
| **数据** | 同源 H5，独立 cache | 同源 H5，独立 cache |
| **状态** | U-Net 训练中 | 启动中 |

两条线共享：物理背景、探头参数、原始 H5 数据源、B-mode 转换标准（REF=64407.58）、损失公式（0.84·SSIM + 0.16·L1）、协作规范。

两条线隔离：代码、cache、GT 派生数据、实验记录、checkpoint、文档中各管各。
