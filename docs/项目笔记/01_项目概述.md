# 01 — 项目概述

## 1. 物理背景

### RCA 探头与正交平面波成像

Row-Column-Addressed (RCA) 阵列由两组正交排列的线阵组成——行阵元和列阵元各 128 个，
阵元沿长轴方向延伸覆盖整个探头边长（25.6 mm）。相比传统全采样矩阵阵列需要 N×N 个通道，
RCA 仅需 2N 个通道即可实现三维成像。

正交平面波（Orthogonal Plane-Wave, OPW）是 RCA 阵列的一种专用发射模式：
行阵元发射、列阵元接收（RC 模式），然后列阵元发射、行阵元接收（CR 模式），
两种模式互补叠加，使点扩散函数（PSF）从各向异性变为各向同性。

### DAS 重建：角度越多画质越好，但越慢

延时求和（Delay-and-Sum, DAS）是超声成像中最经典的重建算法。
发射的倾斜平面波角度越多，相干复合后的图像信噪比和分辨率越高，但计算量随角度数线性增长。

- **3 角度 DAS**（输入角度：3°, 38°, 73°）：帧率高，计算快，但图像散斑粗糙、分辨率差
- **33 角度 DAS**（均匀分布 1°~75°）：画质好，散斑均匀，但计算量是 3 角度的 11 倍

这就是本项目要解决的核心矛盾。

## 2. 项目目标

用深度学习网络从 **3 角度 RF 数据 + 3 角度 baseline B-mode 图像（双输入）**
直接预测接近 **33 角度 DAS 画质的 B-mode 图像**。

- **输入 1**：3 角度的 RF 张量，形状 [1536, 64, 32, 32]
- **输入 2**：3 角度 DAS 的 B-mode 图像，形状 [1, 64, 32, 32]
- **输出**：高质量 B-mode 图像，形状 [1, 64, 32, 32]
- **GT**：33 角度 DAS 的 B-mode 图像（五步标准转换，REF=64407.58）

### 核心创新：RF + 图像双输入融合

传统做法要么只用 RF（channel data），要么只用 B-mode 图像（image-based）。
Hy-PCF 论文（IEEE TMI 2025）证明：
- 纯 RF 模型：有效降噪但过度平滑（oversmooth），丢失散斑纹理
- 纯图像模型：保留散斑但保留噪声，无回声区域有伪影
- **两者融合**：同时降噪和保纹理，各项指标最优

线一已在前序工作中验证了纯 RF 路线存在 oversmooth 问题（TinyBModeRFNet 输出灰雾、Light3DUNet val SSIM 反降）。线二直接走双输入路线——RF 提供丰富的通道声学信息，baseline B-mode 提供结构先验和空间约束，两者互补。

## 3. 和线一的关系

| | 线一（RF_Image_GAN） | 线二（RF_Image_Bmode） |
|---|---|---|
| **探头** | 同一个 RC6gV RCA | 同一个 |
| **平台** | Vantage 256 | 同一个 |
| **输入** | 纯 3 角度 RF | 3 角度 RF + 3 角度 B-mode 图像 |
| **输出** | B-mode 图像 | B-mode 图像 |
| **网络** | 单编码器 U-Net | 双支路编码器 + 融合 |

| **数据源** | 同 1500 H5 | 同 1500 H5（独立 cache） |

## 4. 技术路线

```
线一起点: 纯 RF cGAN（复数空间 + 对抗）
  → Phase 2a 四轮消融：对抗从未通电
  → 路线转向：B-mode 纯监督，去掉对抗
  → TinyBModeRFNet 失败（灰雾，感受野 5 体素）
  → Light3DUNet 训练中（val SSIM 仍在反降）

线二起点: 从线一教训出发
  → 不碰复数空间、不碰对抗
  → 双输入（RF + baseline B-mode）借鉴 Hy-PCF 思想
  → 轻量级双支路 3D U-Net 架构
  → 损失 = 0.84·(1-SSIM3D) + 0.16·L1
```
