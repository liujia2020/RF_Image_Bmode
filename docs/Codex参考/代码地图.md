## 关键约束

| 约束 | 说明 |
|------|------|
| `normalize=True` | cache 生成时三字段同除 scale |
| `restore_scale=True` | reader 端乘回 scale（仅 RF；B-mode 已是 [0,1] 不乘） |
| reader 双输入 | 返回 `input_rf` + `input_bmode` + `label_bmode` |
| 输出 | 单通道 B-mode + sigmoid → [0,1] |
| 损失 | `0.84*(1-SSIM3D) + 0.16*L1`，pytorch-msssim 3D，window=7 |
| BN momentum=0.9 | 防止拼图缝（线一已验证） |
| AMP 关 | 防止 fp16 中间溢出 |
| num_workers=0 | 多进程+memmap WSL 兼容问题 |
| 分层 batch | samples_per_category=2，batch=6 |
| 无对抗/无判别器 | 纯监督路线 |
| 全量验证 | 禁止抽样 |
