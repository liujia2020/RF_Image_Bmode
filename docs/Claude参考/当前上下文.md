### 为什么开线二

用户读到 Hy-PCF 论文（IEEE TMI 2025），其核心发现：
- 纯 channel data（RF）模型 → oversmooth（正与线一症状吻合）
- 纯 image 模型 → 保留噪声/伪影
- **双输入融合**（RF + 单角度图像）→ 降噪且保纹理，各项指标最优

线二直接从这一结论出发：RF + baseline B-mode 双输入，跳过了"验证纯 RF 不够"这个阶段。

### 线二的定位

- 输入：3 角度 RF + 3 角度 baseline B-mode
- 输出：33 角度级别的高质量 B-mode
- 架构：轻量双支路编码器 + 融合（借鉴 Hy-PCF 思想，不照搬 41M 架构）
- 损失：0.84·(1-SSIM3D) + 0.16·L1（纯监督，不碰对抗）
- 训练：WSL / RTX 4060 8G / batch=6
