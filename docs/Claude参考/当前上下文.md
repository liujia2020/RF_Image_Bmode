## 线二是什么(30 秒)

- **目标**:RF + baseline B-mode 双输入 → 高质量 B-mode(借鉴 Hy-PCF,IEEE TMI 2025 的双输入融合思想)。
- **为什么**:线一证明纯 RF 路线 oversmooth(灰雾、丢线靶);Hy-PCF 发现纯 RF→过平滑、纯图像→留噪声、**双输入融合→降噪且保纹理**。线二直接从这个结论起步,跳过"验证纯 RF 不够"。
- **输入**:3 角度 RF + 3 角度 baseline B-mode。**输出**:33 角度级别的高质量 B-mode。
- **损失**:0.84·(1−SSIM3D) + 0.16·L1,纯监督,不碰对抗。
- **运行**:Win11 WSL / RTX 4060 8G。

- 文件:`scripts/probe_dataloader_throughput.py`。
