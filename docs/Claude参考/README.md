## 本项目和线一

线二是 RF_Image 工作区的**第二条独立研究线**。与线一共享：
- 物理背景（同一探头、同一数据采集流程）
- H5 原始数据源
- B-mode 转换标准（REF=64407.58 五步）
- 损失公式（0.84·SSIM + 0.16·L1）
- 协作规范（原则与规范/）

与线一隔离：
- 独立代码仓库（`RF_Image_Bmode` GitHub）
- 独立 cache（WSL 本地路径）
- 独立实验记录
- 线二的 Claude/Codex/Proma 不参与线一事务
